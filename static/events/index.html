從 2012 年底的第零次動員戡亂黑客松開始，雙月一次的百人 g0v 黑客松，竟也已連續辦了兩年。今年，g0v 社群鴨子划水的協作成果逐漸浮出水面。從年初與國際串連，到 318 學運彙集各開源社群的爆發力，各專案因人數參與成長而更多元、更創意。

g0v Summit Unconference 開放工作坊
/ g0v hackath10n

2014/11/09
共筆入口 / Note
提案簡報 / Pitch
中場短講 / Talk
成果報告 / Demo
相關專案

g0v Summit 2014 除了第一天演講型態的研討會，第二天舉行由參與者自主提案的 Unconference。演講、討論、腦力激盪、寫程式皆歡迎。全場共分五個場地，早上全體提案協調後，分軌同時進行。

不同於平常 g0v 黑客松直接動手做的方式，Unconference 特別適合難解複雜問題的討論，讓平時較難聚在一起腦力激盪的各地好手，有辦法聚在一起討論！

Unconference 進行的方式：

  • 動口，也動手，但不需直接做出成果
  • 討論形式不拘，可用電腦共筆協作、白板繪製，甚至白報紙蒐集意見整合結論
  • 在 11/8 起徵求討論主題，主題提出者，討論的帶領者，需要幫助大家進行討論

你可以來：

  • 發起議程
  • 參與討論
  • 聆聽討論
  • 觀察活動
  • 動手實作
